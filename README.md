# Restaurant Bill Splitter

A Streamlit app to scan restaurant receipts using Google Cloud Vision API, assign items to individuals, and export split bills with tax/tip included.

## Features
- Upload and OCR receipts
- Manual edit of OCR output
- Checkbox-based item assignment
- Tax and tip calculation per person
- Export per-person bill as PDF
